SAX 简介

SAX（Simple API for XML）跟 DOM 一样也是用于访问 XML 文档的接口，原先只是 Java 上的 API，后来也在其他语言中有实现版本，如 Python 版的 SAX。详细介绍可参考 IBM 的一篇博文使用 SAX 处理 XML 文档。
