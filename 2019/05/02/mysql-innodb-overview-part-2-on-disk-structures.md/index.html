InnoDB 逻辑存储结构

InnoDB 的逻辑存储结构如下图所示。所有的数据都被逻辑地存放在一个空间中，称之为表空间（tablespace）。表空间又由段（segment）、区（extent）、页（page）组成。

页的本质：

A unit representing how much data InnoDB transfers at any one time between disk (the data files) and memory (the buffer pool).

也就是说，页是磁盘和内存数据传输的单位，默认是16KB。
